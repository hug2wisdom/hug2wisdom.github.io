新的开始

反反复复，折腾来折腾去，最后还是回到了那个最初开始的地方。

回想起来，那个时候一时兴起去搭建自己的博客，已经是 1 年前的事情了，当时也是看到论坛上的人展示自己博客，觉得很好看，很精美。自己就想着能不能拥有一个。

可能是自己内心想成为程序员，所以对有关程序员必备的技能相对的会去多关注一些。然后自己就去搜寻如何去搭建自己的博客，后来逐渐的了解到：Github Page, 域名空间，DNS 之类的，也算是打开了我的新的天空。

接着也接触到 Git, Hexo 等等自己从没有接触的东西，很是兴奋，当时的状态说是废寝忘食，那也不算夸张。搭建博客的时候，因为自己对 Code 了解的甚少，所以就是照着教程，一顿瞎捣鼓，其中参照了无数个教程，可以说是「取百家之长也」。最后在艰难困苦之下，终于搭建好了自己的博客。

搭建完之后，就去尝试着各种主题，可能我这个人有点强迫症，或者说是没有主见，因为在那么多的主题里，没有自己的判断，总是觉得别人说的都是好的，不断的看到新的然后推翻自己刚弄好的，最后竟然越做越差，明明自己喜欢简洁，却搞得花里胡哨，最后不得已 reset all, 导致浪费了很多的时间，还是回到了原点。

这次再重新搭建的时候，说实话，比上次进步了很多。

这次我知道自己的目的是什么，而不是只是看到别人的好看，我也想拥有一个。这一次我是想静下心来，好好的写点文章，好好的记录自己的成长轨迹。其实也想过在微信公众号上书写，但是却感到些许的轻浮，自己的文字和思想的段位不够高，生怕自己的文字分享出去，石沉大海，那样也是自己吃力不讨好。倒不如在自己的小菜园里，安安静静的码点文字，等待自己开花结果的那天。

这一次，在挑选主题的时候，我没有再去挑选那么多令人眼花缭乱的主题，而是见到好的就收手。一直寻找下去，哪里是个头，最后可能也要重蹈上次的覆辙。有句话说的好，选择多了，就是没有选择。有些时候，样式只是锦上添花，最重要的还是内容。就算你的皮囊万般美丽，可是你内心空洞乏味，久了，必然令人审美疲劳，感到厌恶。

这一次还有不同的是，在搜寻教程的时候，我都倾向于 read official document. 不再去四处寻找那些零散的教程，那些充斥着个人咀嚼后残留的味道的教程，直接上英文文档，明明白白，只是需要自己多懂点脑筋。

最近也在努力的恢复以往的英语水平，在努力的学习前端的知识，没想到，这次全部运用上了，一些关于配置的代码我自己也能看懂了，英语我也能读的通顺，能够明白人家字里行间传达的意思了。

我想这是一件很好的事，无形之中，你就在进步。

好了，现在博客已经重新搭建好，一切就像一张白纸一样，等待重新书写，那么就从此时此刻开始，用文字填充这一张白纸，让自己的文字、思想、智慧武装自己，等待开花结果的那一天，满载而归。

BTW，

Wisdom is a new sexy.

加油，努力，有一天写出王小波那样的文字。

总结
