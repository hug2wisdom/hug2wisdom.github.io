JS 运动基础02

通过控制 speed 形成缓冲的视觉。这里需要注意的是 速度要取整数，计算机（浏览器）默认处理 （0 ，1）为 0，（-1， 0） 为 0，所以需要再 speed > 0 的时候，使用 Math.ceil(speed)，在 speed < 0 的时候使用 Math.floor(speed)，使得能达到正确的目的地。
